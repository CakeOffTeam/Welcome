# Working together
 * Fork from KAT repository
 * Clone your repository 

   ``` ruby
     git clone git@github.com:<Your name>/helloWorld.git
   ```
 * Edit README.md 
 * View git status

   ``` bash
     git status
   ```
 * Add your changes

   ``` bash
     git add .
   ```
 * Commit your changes

   ``` bash
     git commit -m "my super explicit message"
   ```
 * Send your changes to your github repo

   ``` bash
     git push origin master
   ```
